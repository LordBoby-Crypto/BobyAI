# BobyAI Roadmap

## Pass 1
Architecture blueprint.

## Pass 2
Frontend app shell for GitHub Pages.

Includes:
- Mobile-first premium UI
- Builder mode selector
- Project request composer
- Settings dialog
- Backend endpoint placeholder
- Build request generator
- GitHub Pages-ready structure

## Pass 3
Cloudflare Worker backend.

Planned:
- Private access code verification
- OpenAI API key stored as a Worker secret
- `/health` endpoint
- `/generate` endpoint placeholder

## Pass 4
Connect frontend to backend.

Planned:
- Send project request to backend
- Receive AI response
- Display generated output

## Pass 5
Project mode system.

Planned:
- Website mode
- Web app mode
- Script mode
- GitHub Pages mode
- ZIP mode

## Pass 6+
Generated files, ZIP download, saved projects, uploads, GitHub repo integration, web search, and local/self-hosted model adapters.
